# Yerevan Public Transport – Security Analysis

Independent research on the security posture of Yerevan’s unified public transport payment system (Telcell, validators, mobile app). Conducted as part of the ISAA Young Professionals Program first‑round assignment (March 2026).

## 🔍 Key Findings

- Validator observed running in **debug mode**, exposing system settings
- Ownership includes **offshore company** (51.33%) and **former prime minister** (16%)
- No enforced lockdown or patch management evident
